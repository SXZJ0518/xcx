## 📋 快速部署检查清单

### 第1步：开通云开发（必须）
- [ ] 打开微信开发者工具
- [ ] 点击顶部「云开发」按钮
- [ ] 点击「开通」，按提示完成开通
- [ ] 复制环境ID（格式：xxx-xxxxx）

### 第2步：配置环境ID（必须）
- [ ] 打开 `app.js` 文件
- [ ] 找到第25行 `cloudEnv`
- [ ] 替换为你的环境ID
```javascript
cloudEnv: '你的环境ID'
```

### 第3步：上传云函数（必须）
- [ ] 右键点击 `cloudfunctions/mall` 文件夹
- [ ] 选择「上传并部署：云端安装依赖」
- [ ] 等待部署完成 ✅
- [ ] 右键点击 `cloudfunctions/initDatabase` 文件夹
- [ ] 选择「上传并部署：云端安装依赖」
- [ ] 等待部署完成 ✅

### 第4步：创建数据库集合（必须）
在云开发控制台 → 数据库中创建：
- [ ] `user`（用户表）
- [ ] `product`（商品表）
- [ ] `category`（分类表）
- [ ] `order`（订单表）
- [ ] `address`（地址表）
- [ ] `banner`（轮播图表）

### 第5步：初始化数据（推荐）
- [ ] 云开发控制台 → 云函数
- [ ] 找到 `initDatabase`
- [ ] 点击「测试」→「调用测试」
- [ ] 点击「调用」
- [ ] 查看返回结果 ✅

### 第6步：配置数据库权限（必须）
在云开发控制台 → 数据库 → 权限设置：
- [ ] `product`、`category`、`banner`：所有用户可读，仅管理员可写
- [ ] `user`、`order`、`address`：所有用户可读，仅创建者可写

### 第7步：测试小程序
- [ ] 在微信开发者工具中编译运行
- [ ] 查看首页是否显示商品
- [ ] 测试商品详情、购物车等功能

## 💡 常见问题

### Q: 云函数上传失败怎么办？
A: 检查：
1. 云开发是否已开通
2. 网络连接是否正常
3. 代码是否有语法错误

### Q: 首页没有商品显示？
A: 确保：
1. 已运行 `initDatabase` 云函数
2. 数据库中已有商品数据
3. 数据库权限配置正确

### Q: 想要自己添加商品？
A: 可以通过以下方式：
1. 在云开发控制台的数据库中手动添加
2. 后续可以开发管理后台界面
